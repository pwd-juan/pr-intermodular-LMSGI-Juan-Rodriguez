## Estructura
Lo único que he hecho diferente a la estructura que solicitaba el enunciado en la galería de servicios. En el cual, he decidido hacer un bento grid sencillo.

## Dificultades
Cuando estuve diseñando las páginas lo realice en pantalla dividida (en un lado de la pantalla la página y en el otro el editor), esto produjo que al pasar a pantalla completa algunos contenidos se vieran vacíos. Una de las soluciones que he encontrado es poner anchos máximos (max-width). En algunos de estos he utilizado pixeles por que no le encontrar sentido utilizar metidas relativas.

Otra dificultad ha sido hacer responsive el aside de la página proyecto estrella. La solución encontrada ha sido usar un display grid.

## Markdown
Este proyecto también sirve para apuntes de Markdown. El cual es un lenguaje de marcas, es sencillo y permite escribir documentación de forma rápida y mantener un estilo visual. Las etiquetas de Markdown son:

* `#` Para los títulos, cuantos más `#` más baja el nivel de relevancia del título.
``` [Markdown]
## Título h2
```

* Los saltos de línea son con doble espacio.

* Las citas son usando `>`.
``` [Markdown]
> En seguridad informática, siempre se dice desde el principio que el sistema informático más seguro es uno apagado, sin conexión y encerrado en una habitación sin llave. Es decir, uno inútil.
```

* Envolviendo una palabra con `*` puedes obtener palabra en cursiva, y con `**` las consiguen en negrita.
``` [Markdown]
*cursiva*
**negrita**
```

* Con el acento grave (el invertido) escribes código, apilando tres de ellos puedes crear un marco de código en bloque, y con `[]` seleccionas el lenguaje. (No puedo poner el símbolo porque lo detecta como código)

``` [Markdown]
    `Código en linea`

    ``` [lenguaje]
    Código en 
    bloque
    ```
```

* Para las listas puedes utilizar `*` para lista no ordenadas y números para listas ordenadas.
``` [Markdown]
    * Elemento 1 de una lista no ordenada.
    * Elemento 2 de una lista no ordenada.

    1. Elemento 1 de una lista ordenada.
    2. Elemento 2 de una lista ordenada.
    3. Elemento 3 de una lista ordenada.
```

* Los enlaces tiene una estructura más complicada, combina `[]` y `()`.
``` [Markdown]
    [Texto del enlace aquí]("Dirección del enlace")
```

* Para las imágenes es la misma estructura que los enlaces añadiendo un `!` al inicio.
``` [Markdown]
    ![Texto alternativo]("Dirección de la imagen")
```

* Para las tablas es usando `|`. Con `:` puedes elegir la justificación del texto, `:---` derecha, `---:` izquierda y `:---:` centro. 
``` [Markdown]
    | Syntax      | Description |
    | ----------- | ----------- | < Esta línea es necesaria
    | Header      | Title       |
    | Paragraph   | Text        |
```

* Más elementos de markdown en [markdownguide.org](https://www.markdownguide.org/cheat-sheet/).
